# Geobattery — 3-D thermal models

Interactive 3-D temperature fields from a full-3D conjugate-heat-transfer study of seasonal
heat storage in the flooded **Reiche Zeche** research drift (TU Clausthal, MOVIE project).

**Live viewer:** https://aliahmadi-tuc.github.io/geobattery-3d/

Models (glTF, coloured 10–80 °C):
- `models/drift_charge.gltf` — as-built surveyed drift, year-1 end of charge (the injected plume).
- `models/array_year10.gltf` — 10 m-pitch array unit cell, year-10 end of storage (heat retained).

Companion to the ALERT Geomaterials 2026 poster *“Geobattery: a dimensionless criterion for
seasonal heat storage in flooded mine workings.”* The geometry is the real surveyed drift; the
duty cycle is hypothetical.

Authors: **Ali Ahmadi**, Eleni Gerolymatou.
